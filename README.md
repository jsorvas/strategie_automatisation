# Stratégie d'automatisation
## Framework ATAP 

L'automatisation peut se dérouler en 4 étapes 

1) Analyser
2) Traduire
3) Automatiser
4) Prouver


Analyser :
Ce qu'il faut tester, les données, et le résultat attendu

Traduire :
Commenter les étapes

Automatiser :
Ecrire le script automatisé sous chaque commentaire 

Prouver :
Vérifier les actions (clic, saisie, texte) avec une assertion qui le confirme
